Standard Medications
NS 0.9% 500 mL × 1 — WHITE spike
Heparin 10,000 U in 500 mL NS = 20 U/mL — ORANGE spike
Methoxsalen (Uvadex) 200 mcg / 10 mL · single-use vial · discard remainder
Methoxsalen dose = ECV × 0.017 mL
Pre-Procedure Checklist
CBC with Hct ≤ 48 hours old (Therakos label requirement)
Patient education: UV precautions × 24 h post — sunglasses, long sleeves, indoors
ECV %TBV: < 10% = standard · 10–15% = caution · > 15% adult / > 10% peds < 30 kg = blood prime
Lamp life > 10 hours remaining (18-lamp assembly, 150 hr total life)
Peripheral IV: hot packs / tourniquet to optimize flow; central line if peripheral inadequate

Machine Setup (follow laminated guide)
1. Establish Ready State — check operator interface for "13 zeros"
2. Insert photoactivation module, remove kit, hang bags, insert pump tubing organizer
3. Install drive tube / centrifuge bowl — turn clockwise to secure
4. Load pump tubing segments — install from the bottom up
5. Smart Card → reads kit lot/expiration
6. Spike fluids: NS (WHITE) + Heparin (ORANGE)
7. Cassette install order: C-R-Y — Clear, Red, Yellow — then bowl
8. Air detectors, Hct cuvette, pressure domes — verify each click into place
9. Pressure pods LAST — spread tabs firmly, verify membrane intact (no green caps remaining)
10. Hang patient lines on the IV pole; close Collect Line (RED), clean injection port
Machine Programming — verify "13 zeros"
Return Bag Threshold (RBTV) = 100 mL (institutional default — do not exceed)
Max speed collect = 50 mL/min
Max speed return = 50 mL/min
Collect Pressure Limit = 200 mmHg
Return Pressure Limit = 200 mmHg
Total blood processed = 1500 mL max
Bowl Optic Threshold default = 140 mL (raise/lower by 10 if interface alarms — see Alarm #45)
Priming & Testing (auto ≈ 7 minutes)
1. Press "Prime Access"
2. Select Single or Double needle to match access setup
3. Connect Blue → Red when prompted
4. System auto-primes and runs Prime Test (~7 min). Resolve any prime test failures before proceeding.
Starting Treatment & Phase Sequence
Start slow: Collect 25 mL/min, Return 10 mL/min
COLLECT Phase 1: Purging air
COLLECT Phase 2: Drawing / Recirculating
When you hear BEEP, OR NET balance ≤ −100, OR blood processed > 300 mL → increase: Collect 30–40, Return 35–45
BUFFY COAT phase: Open Graph → watch S-wave (Hct rises) → look for "END BUFFY" prompt
Full sequence: COLLECT → BUFFY → UVA → PAUSE → ELUTRIATE → EMPTY BOWL → RINSE
Inject Uvadex into photoactivation bag per CELLEX prompt (before UVA cycle starts)
Interrupting Treatment — PAUSE vs STOP
PAUSE (up to 10 min): flush access · give saline bolus · go to SETUP · go to BUFFY COAT — all preserve treatment
STOP: END TREATMENT, ABORT TREATMENT, or go to SETUP — these terminate the treatment
⚠ Centrifuge bowl re-prime: resuming collection after centrifuge stops requires re-prime → displaces leukocytes already collected → ECV runs higher than anticipated
⚠ PHOTOACTIVATE must run to 00:00 — interrupting before completion means calculated UVA dose is NOT delivered
Alarms — MUTE → Resolve → RESET → START
Process: Press MUTE (silences ≤ 60 sec) → Resolve per Section 6 → Press RESET → Press START
Alarm types: LOW (operator-correctable) · MEDIUM (technical, needs investigation) · No HIGH-priority alarms exist on CELLEX
Common alarms: #41 Air Detector · #45 Red Blood Cell Pump (interface high/low) · #66 Anticoagulant bag empty · #11 Photoactivation Chamber Door
Alarm #45 — RBC Pump (interface position):
  • Interface too LOW: SETUP → LOWER Bowl Optic Threshold by 10 → SAVE → START
  • Interface too HIGH: RESET → STOP → START (purges bowl and re-establishes interface)
Bowl Optic Sensor & Specific Gravity
Default Bowl Optic Threshold = 140 mL (adjustable in SETUP)
May raise threshold (e.g. 150 mL) if patient's Hct is unusually low and plasma/RBC interface is too low
Specific gravity order (top → bottom of bowl): Platelets (1.040) · Lymphocytes (1.055–1.065) · B cells / Promyelocytes · Myelocytes / Basophils · Reticulocytes · Band + Segmented Neutrophils · Erythrocytes (1.078–1.114) · Aged erythrocytes
Total centrifugal force = function of bowl rotation; do not stare directly into the laser sensor beam
Maintenance Schedule
After each use: System exterior · Photoactivation chamber door · Air detectors · Pressure sensors · Hematocrit sensor · Leak Detector (Centrifuge Well + Vinegar) — clean with 1:10 bleach or approved disinfectant wipe
Daily: Centrifuge chamber (1:10 bleach), Bowl Optic Lens (dry the lens with a dry, lint-free cloth — clean and streak-free)
As needed: Operator interface (mild soap + warm water), Mallinckrodt service for fluid leaks
Annually: Replace 9V battery (powers low-battery alarm during power loss)
Lamp assembly: 18 lamps, 150 hr total life — replace when ≤ 10 hr remaining. Keep old lamp set as backup if new fails.
Patient Safety & Special Considerations
UV precautions × 24 hr post-treatment: Wear UV-blocking sunglasses, long sleeves, hats; stay indoors when possible — methoxsalen photosensitivity persists
GVHD patients — DVT/PE risk: Increased thromboembolism reported with UVADEX in GVHD population per Therakos labeling. Consider extra anticoagulation per physician.
Pediatric / < 30 kg: Use 10% TBV ceiling (not 15%). Blood prime if patient's Hct or weight can't bring ECV under threshold (see Operator's Manual page 1-3).
Raising Hct may avoid blood prime — transfusion to Hct ≥ 30% can pull ECV into safe range per Table 1
Hct < 27%: Off-chart — patient cannot undergo standard procedure. Optimize Hct or use blood prime protocol.
Citrate / heparin tolerance: Monitor ionized calcium, BP, mental status during run. Calcium replacement protocol per apheresis standing orders.
